> Command line utility for quickly proxying of a given url.

## Install
```bash
$ yarn global add @pascaliske/bsproxy
```

## CLI
Use the globally installed cli to quickly open up a bsproxy:

```bash
$ bsproxy <url> [<custom-port>]
```
